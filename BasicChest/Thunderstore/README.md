# Basic Chest

Adds Basic Chest, chest that contains a selection of "basic" items, such as stat increases or unconditional damage boosts. Item list and item weights can be configured in the config. Chest starts at 40 gold (configurable), spawns on the same stages as Large Chest, has the same chance to spawn.

Default item list:
* Tier 1 (79.2 weight)
  * Lens-Maker's Glasses, Paul's Goat Hoof, Energy Drink, Soldier's Syringe, Armor-Piercing Rounds, Sticky Bomb, Backup Magazine
* Tier 2 (19.8 weight)
  * Hopoo Feather, AtG Missile Mk. 1, Ukulele, Will-o'-the-wisp, Kjaro's Band, Runald's Band  
* Tier 3 (1 weight)
  * Brilliant Behemoth, Alien Head, Hardlight Afterburner, Ceremonial Dagger
