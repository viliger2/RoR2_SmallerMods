# Forlorn Wreckage Stage 5

Moves Forlorn Wreckage to Stage 5 with some changes to make it suitable as stage 5.

## Changes

* Stage is now stage 5 with appropriate changes to destinations.
* Access Node (Solus path thing) is disabled with config option to re-enable it.
* Monsters
  * Added Parent, Grandparent and Elder Lemurian to spawn pool
* Interactables
  * Increased interactable credit pool to 520
  * Added Tier 2 Tri-shop
  * Removed Tier 1 Tri-shop
  * Swapped teleporter with lunar teleporter
  * Added artifact portal with 3 new jump pads that lead to it. Good luck getting down.
